<!-- Plese follow these step -->
1. cd backend
2. cp .env.example .env -> edit with your MongoDB Atlas URI and JWT secret
3. npm install
4. npm run dev

## Quick start (frontend)
1. cd frontend
2. npm install
3. npm run dev

Front-end expects API at http://localhost:5000/api by default.
